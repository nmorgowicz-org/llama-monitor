## Critical Bug Fixes

### 1. Export Menu Auto-Hide Bug
**File:** `static/js/features/chat-params.js:609`

**Issue:** A document-level click listener was hiding the export menu on EVERY click, breaking the export flow.

```javascript
// BEFORE (line 609)
document.addEventListener('click', () => document.getElementById('chat-export-menu')?.classList.add('hidden'));

// AFTER - Removed entirely
```

### 2. Modal Navigation State Broken
**Files:** `static/js/features/nav.js`, `static/js/features/settings.js`, `static/js/features/models.js`, `static/js/features/sessions.js`, `static/js/compat/globals.js`

**Issue:** Settings/Sessions/Models buttons used `id` attributes instead of `data-tab`, so `initNav()` never bound click handlers to them.

**Fix:**
1. Added `data-tab` attributes to HTML buttons
2. Updated `nav.js` to handle modal tabs (no page div):
   ```javascript
   // In switchTab(), check for page div first
   if (!page) {
       // Toggle button active state
       // Call modal open function via window
       if (name === 'settings') window.openSettingsModal?.();
       // ... etc
       return;
   }
   ```
3. Removed duplicate bindings from settings.js, models.js, sessions.js
4. Exposed modal open functions on `window` in globals.js
